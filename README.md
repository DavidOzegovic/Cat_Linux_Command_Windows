# Cat_Linux_Command_Windows
USAGE: .\cat [file_name]

This Project is a simple recreation of one of the most famous Linux commands, "Cat".
I re-created it cause i wanted a fun project that gonna have value to me.
I'm gonna update this project: add more features(more flags)

During the development of this project, I learned:

How to use command-line arguments (argc and argv) to accept options and file names from the user.
How to open, read, and close text files using fopen(), fgetc(), and fclose().
How to handle errors, such as when a file does not exist or cannot be opened.
How to implement features similar to the Linux cat command:
-n to display line numbers,
-E to show a $ character at the end of each line,
-A to display tab characters as ^I.
How to use Boolean variables to track the state of the program.
How to work with special characters such as newline (\n) and tab (\t).
How to detect the beginning and end of lines while processing text files.
How to organize program logic using loops, conditional statements, and flags.
